Get Started Today

The first application we created was an inventory management system.

Ever since, we’ve been working to make sure that clients can stay connected with their inventory, products and processes.

Here’s how we got started…

A small local company that received motorcycles from manufacturers for dealers, and then prepped them for the showroom, needed a way to track that they had received the motorcycles, and then know when the dealer needed them and they were delivered.

They were currently doing this in a MS Access 97 database (From MS Access 97 to Mobile friendly web application with an enterprise level database) The company was starting to grow and they needed to track more and more things, including invoicing, shipping, and they needed a way for the dealers to view their inventory and select the units that they wanted delivered.

This application is now a full ASP.NET MVC web application with SQL Server that has a section for the company employees to interact with the data and process that they need as well as the dealers and manufacturers.
